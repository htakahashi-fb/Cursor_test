# 日報タスク分析ダッシュボード

日報のMarkdownファイルを解析し、タスクの傾向を可視化するWebダッシュボードです。

## 機能

### 1. データ解析
- 日報Markdownファイルの自動解析
- 時間帯別タスクの抽出
- タスクのカテゴリ自動分類

### 2. 分析機能
- **タスク頻度分析**：出現回数の多いタスクをランキング表示
- **時間帯別作業量**：各時間帯の作業量を棒グラフで表示
- **カテゴリ別分析**：カテゴリごとの作業時間・出現回数を円グラフ・棒グラフで表示
- **時系列分析**：日付ごとのタスク数・作業量の推移を折れ線グラフで表示

### 3. UI機能
- ドラッグ&ドロップでファイルをアップロード
- インタラクティブなグラフ（Chart.js使用）
- タスクのカテゴリ別フィルタリング
- レスポンシブデザイン（モバイル対応）

## 使用方法

1. `index.html` をブラウザで開きます
2. 日報のMarkdownファイル（`.md`）をドラッグ&ドロップするか、クリックしてファイルを選択します
3. 複数の日報ファイルを同時にアップロードできます
4. アップロード後、自動的に分析結果が表示されます

## 対応している日報形式

以下の形式の日報に対応しています：

```markdown
# 日報 YYYY/MM/DD (曜日) 名前

## 本日の業務内容
- **時間**：
  - 09:00～10:00：タスク内容1。タスク内容2。
  - 10:00～12:00：タスク内容3
  ...
```

- タイトル行から日付を抽出します（`# 日報 YYYY/MM/DD` 形式）
- 時間帯は `HH:MM～HH:MM` 形式で記述してください
- タスクは「。」「、」で区切って複数記述できます

## タスクのカテゴリ分類

以下のキーワードに基づいて自動的にカテゴリ分類されます：

- **会議・ミーティング**：ミーティング、会議、打ち合わせ、スタンドアップ、デイリー、週次、月次、商談、アポイント
- **資料作成**：資料、スライド、提案、報告書、手順書、ドキュメント、作成、整備
- **開発・デプロイ**：開発、デプロイ、バグ、修正、対応、実装、コード、テスト、動作確認
- **顧客対応・商談**：顧客、クライアント、商談、ヒアリング、問い合わせ、対応、提案、リサーチ
- **その他**：上記のいずれにも該当しないタスク

## 注意事項

- ファイルはブラウザ上で処理されるため、データはサーバーに送信されません
- 大量のファイルをアップロードする場合は、ブラウザのパフォーマンスに影響する可能性があります
- 日報ファイルは `.md` 拡張子である必要があります
